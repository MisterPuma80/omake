# Omake

Omake is a [Blazium](https://github.com/blazium-games/blazium) module with faster C++ scene tree functions.

# Install

```bash
git clone -b blazium-4.5 https://github.com/blazium-games/blazium
git clone https://github.com/MisterPuma80/omake blazium/modules/omake
./blazium/modules/omake/install_module_hooks.sh
```
